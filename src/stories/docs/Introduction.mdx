# Arkite UI

Production-ready React component library for SaaS admin panels.

## Features

- **55+ components** — Buttons, Forms, DataTable, Modal, Drawer, Toast, and more
- **Theme system** — HSL CSS variables, 4 presets, `createTheme()` from hex colors
- **Dark mode** — Class-based toggle with full token support
- **Tailwind CSS v4** — CSS-first configuration with `@theme` tokens
- **Accessible** — Built on Radix UI primitives with ARIA support
- **TypeScript** — Full type safety with exported interfaces
- **Tested** — 111+ unit tests with Vitest

## Quick Start

```bash
pnpm add @arkite/ui
```

```tsx
import { Button } from '@arkite/ui'
import '@arkite/ui/styles.css'

<Button variant="primary">Get Started</Button>
```
