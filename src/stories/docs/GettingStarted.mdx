# Getting Started

## Installation

```bash
npm install @arkite/ui
```

### Peer Dependencies

```bash
npm install react react-dom tailwindcss zustand lucide-react
```

Optional (for animated components):

```bash
npm install framer-motion
```

## Setup

### 1. Import CSS

In your app's entry point (e.g. `main.tsx` or `layout.tsx`):

```tsx
import '@arkite/ui/styles.css'
```

This includes all theme tokens, base styles, and utility classes.

### 2. Tailwind CSS v4

Arkite UI uses **Tailwind CSS v4**. Your project's CSS file should import Tailwind:

```css
@import "tailwindcss";
@import "@arkite/ui/styles.css";
```

### 3. Use Components

```tsx
import { Button, Input, Badge, DataTable } from '@arkite/ui'

function App() {
  return (
    <Button variant="primary" onClick={() => alert('Hello!')}>
      Click me
    </Button>
  )
}
```

## TooltipProvider

If you use any Tooltip components, wrap your app with `TooltipProvider`:

```tsx
import { TooltipProvider } from '@arkite/ui'

function App() {
  return (
    <TooltipProvider>
      {/* your app */}
    </TooltipProvider>
  )
}
```

## Local Development (npm link)

For consuming projects during local development:

```bash
# In arkite-ui directory
npm link

# In your project
npm link @arkite/ui
```
