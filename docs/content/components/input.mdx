---
title: Input
description: Text input field with size variants, addons, and error state.
---

## Import

```tsx
import { Input } from '@arkite/ui'
```

## Usage

```tsx
<Input placeholder="Enter your name" />
<Input size="sm" />
<Input size="lg" />
<Input error errorMessage="This field is required" />
<Input disabled />
<Input leftAddon="https://" rightAddon=".com" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Input size |
| `error` | `boolean` | `false` | Error border style |
| `errorMessage` | `string` | - | Error text below input |
| `leftAddon` | `ReactNode` | - | Left addon content |
| `rightAddon` | `ReactNode` | - | Right addon content |
| `fullWidth` | `boolean` | `true` | Stretch to container width |

Extends all native `<input>` HTML attributes.
