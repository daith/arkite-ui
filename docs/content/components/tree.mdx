---
title: Tree
description: Hierarchical tree view with expand/collapse, selection, and checkboxes.
---

## Import

```tsx
import { Tree, type TreeNode } from '@arkite/ui'
```

## Usage

```tsx
const data: TreeNode[] = [
  {
    key: 'admin',
    label: 'Admin',
    children: [
      { key: 'users', label: 'Users' },
      { key: 'settings', label: 'Settings' },
    ],
  },
]

<Tree data={data} defaultExpandedKeys={['admin']} />
```

## With Selection

```tsx
const [selected, setSelected] = useState<string>()

<Tree
  data={data}
  selectedKey={selected}
  onSelect={(key) => setSelected(key)}
/>
```

## Checkable (Permissions)

Parent-child cascading: checking a parent checks all children, and partial selection shows an indeterminate state.

```tsx
const [checked, setChecked] = useState<string[]>([])

<Tree
  data={permissions}
  checkable
  checkedKeys={checked}
  onCheckChange={setChecked}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | `TreeNode[]` | Required | Tree structure |
| `defaultExpandedKeys` | `string[]` | `[]` | Initially expanded nodes |
| `expandedKeys` | `string[]` | - | Controlled expanded state |
| `onExpandChange` | `(keys: string[]) => void` | - | Expand callback |
| `checkable` | `boolean` | `false` | Show checkboxes |
| `checkedKeys` | `string[]` | - | Checked node keys |
| `onCheckChange` | `(keys: string[]) => void` | - | Check callback |
| `selectedKey` | `string` | - | Selected node key |
| `onSelect` | `(key: string, node: TreeNode) => void` | - | Selection callback |
| `showLines` | `boolean` | `false` | Show connecting lines |

## TreeNode

| Prop | Type | Description |
|------|------|-------------|
| `key` | `string` | Unique identifier |
| `label` | `ReactNode` | Display content |
| `children` | `TreeNode[]` | Child nodes |
| `disabled` | `boolean` | Disable interaction |
| `icon` | `ReactNode` | Custom icon |
