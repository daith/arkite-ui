---
title: Toast
description: Notification toasts with imperative API. No context provider needed.
---

## Import

```tsx
import { toast, ToastContainer } from '@arkite/ui'
```

## Setup

Add `<ToastContainer />` once in your app's root layout:

```tsx title="src/app/layout.tsx"
import { ToastContainer } from '@arkite/ui'

export default function Layout({ children }) {
  return (
    <>
      {children}
      <ToastContainer />
    </>
  )
}
```

## Usage

Call `toast` methods from anywhere — no hooks or context needed:

```tsx
import { toast } from '@arkite/ui'

// Basic variants
toast.success('Changes saved')
toast.error('Something went wrong')
toast.warning('Disk space low')
toast.info('New version available')

// With description
toast.success('User created', {
  description: 'An invitation email has been sent.',
})

// With action
toast.error('Failed to save', {
  action: { label: 'Retry', onClick: () => handleRetry() },
})

// Loading → dismiss
const id = toast.loading('Uploading...')
await upload()
toast.dismiss(id)
toast.success('Upload complete')

// Dismiss all
toast.dismissAll()
```

## Custom Duration

```tsx
toast.info('This stays for 10 seconds', { duration: 10000 })
```

Default duration is 5000ms (5 seconds).

## API Reference

### `toast` methods

| Method | Returns | Description |
|--------|---------|-------------|
| `toast.show(title, options?)` | `string` (id) | Show default toast |
| `toast.success(title, options?)` | `string` (id) | Green success toast |
| `toast.error(title, options?)` | `string` (id) | Red error toast |
| `toast.warning(title, options?)` | `string` (id) | Yellow warning toast |
| `toast.info(title, options?)` | `string` (id) | Blue info toast |
| `toast.loading(title, options?)` | `string` (id) | Toast with spinner |
| `toast.dismiss(id)` | `void` | Remove specific toast |
| `toast.dismissAll()` | `void` | Remove all toasts |

### ToastContainer Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `position` | `'top-right' \| 'top-left' \| 'bottom-right' \| 'bottom-left'` | `'bottom-right'` | Toast position |
| `max` | `number` | `5` | Maximum visible toasts |
